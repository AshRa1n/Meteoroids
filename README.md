# Meteoroids
Asteroids-like game

В данной top-down игре вы управляете кораблем, который может двигаться вперед (W,arrowUp), а так же вращаться в стороны (A/D,arrowLeft,arrowRight). Корабль умеет стерлять (ЛКМ). На сцене может находится не более 4 выстрелов одновременно. Ваша задача уничтожить противников, появляющихся на поле: Метеоры, Большие НЛО, Маленькие НЛО
После уничтожения всех противников, появляется следующая волна. В начале вам будут встречаться только метеориты
При попадании по метеориту, он раскалывается на 2 более маленьких, но более быстрых осколка. При столкновлении с метеоритом или осколком игрок теряет одну жизнь
Большие НЛО движутся по прямой и стреляют в случайном направлении. 
Маленькие НЛО движутся по прямой пока рядом не окажется игрок. Они меняют направление в сторону игрока и начинают стрелять.
При попадании/столкновении с нло игрок теряет жизнь
Если игрок потеряет все 3 жизни - игра закончится
При достижении конца экрана, любой объект перемещается в противоположный по обеим осям (x,y) край карты

На данный момент игра работает правильно только на разрешении 1920х1080
